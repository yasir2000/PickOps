# DDDOps - Domain-Driven Design Operations

DDD architecture tooling, bounded context management, and strategic design automation.

## 🎯 Overview

- Bounded context visualization (Context Mapper)
- Event storming tools
- Aggregate design patterns
- Strategic design documentation
- Domain model validation
- Context mapping automation

## 📦 Components

| Service | Port | Description |
|---------|------|-------------|
| Context Mapper | 8080 | DDD modeling tool |
| EventStorming | 3000 | Collaborative modeling |
| Miro Board | 8081 | Visual collaboration |
| Structurizr | 8082 | Architecture diagrams |
| PlantUML | 8083 | Diagram generation |
| Domain Story | 8084 | Domain storytelling |

## 🚀 Quick Start

```bash
cd DDDOps
cp .env.example .env
./scripts/start.sh
```
