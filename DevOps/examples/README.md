# DevOps Examples

Real-world CI/CD pipeline examples and automation workflows.

## Examples

1. [jenkins-pipeline](./jenkins-pipeline/) - Complete CI/CD pipeline with Jenkins
2. [docker-build-push](./docker-build-push/) - Build and push Docker images to Harbor
3. [gitlab-ci-cd](./gitlab-ci-cd/) - GitLab CI/CD with runners
4. [sonarqube-scan](./sonarqube-scan/) - Code quality scanning
5. [vault-secrets](./vault-secrets/) - Secret management with Vault

## Quick Start

```bash
# Start DevOps stack
cd ../
docker-compose up -d

# Access services
# Jenkins: http://localhost:8080
# Harbor: http://localhost:8081
# SonarQube: http://localhost:9000
```
