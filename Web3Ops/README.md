# Web3Ops - Web3 Operations

Web3 infrastructure, DApp deployment, and decentralized services.

## 🎯 Overview

- Web3 gateways (Infura alternative)
- DApp hosting (IPFS, Arweave)
- Wallet services
- DeFi protocol integration
- NFT infrastructure
- DAO tooling

## 📦 Components

| Service | Port | Description |
|---------|------|-------------|
| IPFS | 5001 | Decentralized storage |
| Ceramic | 7007 | Decentralized data |
| Lit Protocol | 7470 | Access control |
| The Graph | 8020 | Data indexing |
| Lens Protocol | 8080 | Social graph |
| ENS | 8545 | Name service |

## 🚀 Quick Start

```bash
cd Web3Ops
cp .env.example .env
./scripts/start.sh
```
