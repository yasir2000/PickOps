## 🚀 Quick Start

### Start Any Stack (3 Simple Steps)

```bash
# 1. Navigate to any stack
cd HealthOps

# 2. Create environment file
cp .env.example .env

# 3. Launch the stack
docker-compose up -d
```

### Using Management Tools

```bash
# Using the management script
./manage.sh healthops start
./manage.sh bankops logs
./manage.sh eduops restart

# Using Make
make start STACK=agriops
make logs STACK=mediops
make stop STACK=enerops
```
