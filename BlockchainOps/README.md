# BlockchainOps - Blockchain Operations

Blockchain node management, monitoring, and development platform.

## 🎯 Overview

- Blockchain nodes (Ethereum, Bitcoin, Hyperledger)
- Block explorers
- Smart contract development
- Network monitoring
- Validator management
- Transaction analysis

## 📦 Components

| Service | Port | Description |
|---------|------|-------------|
| Geth | 8545 | Ethereum node |
| Besu | 8546 | Enterprise Ethereum |
| IPFS | 5001 | Distributed storage |
| Ganache | 8545 | Test network |
| Blockscout | 4000 | Block explorer |
| The Graph | 8020 | Indexing protocol |

## 🚀 Quick Start

```bash
cd BlockchainOps
cp .env.example .env
./scripts/start.sh
```
